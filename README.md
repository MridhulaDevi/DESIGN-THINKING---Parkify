# Parkify – Smart Parking Management System

## Problem Statement
Finding available parking spaces in urban areas is time-consuming and often leads to traffic congestion, fuel wastage, and inefficient utilization of parking resources. Traditional parking systems lack real-time visibility, centralized management, and convenient booking facilities.

## Solution
Parkify provides a digital platform that enables administrators and parking owners to manage parking locations, monitor slot availability, update pricing, track bookings, and improve parking space utilization through an intuitive dashboard.

## Features
- Admin and Parking Owner Login
- Parking Location Management
- Slot Management and Visualization
- Booking Management
- Dynamic Pricing Updates
- Parking Open/Close Status Control
- Dashboard Analytics and Reports
- MySQL Database Integration
- Cloud Database Support (Clever Cloud)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot (Java)
- Database: MySQL (Clever Cloud)
- Version Control: Git & GitHub
